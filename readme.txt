Main Program: demo_viper.m

To get our paper's CMC result, choose runs_mode='repeat' to use the same training/testing splits
Otherwise, choose runs_mode='new' to get new random splits.

Note: This code is built on the public code and data provided by KISSME paper:
Kostinger, M., Hirzer, M., Wohlhart, P., Roth, P.M., and Bischof H.:
 "Large scale metric learning from equivalence constraints", 
IEEE Conf. Computer Vision and Pattern Recognition, 2012, pp. 2288¨C2295.

Citation for using this code: 
Mingyong Zeng, Zemin Wu, Chang Tian,etc,"Person re-identification based on 
a novel mahalanobis distance feature dominated KISS metric learning," 
Electronics Letters (manuscript in review).
